## 🇰🇷 한국어

### 이번 업데이트 (v1.0.4)
- **CCH(캐릭터 체인지 헬퍼) 세트 복사/저장 방식을 세트 단위로 개선**
  - `save` / `copy`가 이제 **현재 보고 있는 세트 1개**에만 동작합니다. (예전엔 여러 세트가 함께 저장/덮어써짐)
  - 저장 목록이 **`캐릭터명 (직업) Set N`** 형태로 표시되어, 어느 세트를 저장했는지 한눈에 확인됩니다.
  - Set1 / Set2 / Set3을 **각각 별도로 저장**할 수 있고, 서로 덮어쓰지 않습니다.
  - `copy`는 선택한 세트에만 적용되고, 나머지 세트는 그대로 유지됩니다.
  - 아에테르 젬(1~4)도 세트별로 정확히 저장/복사됩니다.
  - 예전 버전에서 저장해둔 항목은 자동으로 **"Set 1"로 이관**됩니다. (불필요하면 delete 메뉴에서 삭제)

### 설치 방법
- **기존 넥서스 애드온(nexus_addons ipf 파일)만 삭제**하고 이 애드온을 넣어주세요.

### 사용법
- `Ctrl + ` `(백틱) : 토글창 열기/닫기
- Zmei 하드 레이드 참가 버튼 클릭 시 물약 자동 교체

## 🇺🇸 English

### What's New (v1.0.4)
- **Reworked CCH (Character Change Helper) copy/save to be per-set**
  - `save` / `copy` now act on **only the currently shown set**. (Previously multiple sets were saved/overwritten together.)
  - Saved sources are listed as **`Name (Job) Set N`**, so you can see exactly which set was saved.
  - Set1 / Set2 / Set3 can each be **saved separately** without overwriting one another.
  - `copy` applies only to the selected set; your other sets are left intact.
  - Aether gems (1–4) are now saved/copied correctly per set.
  - Sources saved by older versions are automatically **migrated to "Set 1"** (delete them from the delete menu if unwanted).

### How to Install
- **Remove only your old nexus_addons ipf file**, then add this addon.

### Usage
- `Ctrl + ` ` (backtick): open/close the toggle window
- Potions auto-swap when you click the Zmei Hard raid join button

## 🇯🇵 日本語

### 今回のアップデート (v1.0.4)
- **CCH（キャラクターチェンジヘルパー）のコピー/保存をセット単位に改善**
  - `save` / `copy` が**現在表示中のセット1つ**だけに作用するようになりました。（以前は複数セットがまとめて保存/上書きされていました）
  - 保存リストが **`キャラ名 (職業) Set N`** の形式で表示され、どのセットを保存したか一目で分かります。
  - Set1 / Set2 / Set3 を**それぞれ個別に保存**でき、互いに上書きしません。
  - `copy` は選択中のセットだけに適用され、他のセットはそのまま維持されます。
  - エーテルジェム（1〜4）もセットごとに正しく保存/コピーされます。
  - 旧バージョンで保存した項目は自動的に **「Set 1」へ移行**されます。（不要なら delete メニューから削除）

### インストール方法
- **古い nexus_addons の ipf ファイルだけを削除**して、このアドオンを入れてください。

### 使い方
- `Ctrl + ` `（バッククォート）: トグルウィンドウの開閉
- Zmei ハードレイド参加ボタンを押すとポーションが自動で切り替わります
