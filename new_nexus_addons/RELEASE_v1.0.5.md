## 🇰🇷 한국어

### 이번 업데이트 (v1.0.5)
- **Other Character Skill List(다른 캐릭터 스킬 목록) 설정 프레임이 안 열리던 버그 수정**
  - 관리 목록의 ⚙(Call Settings Frame) 버튼을 눌러도 창이 안 뜨던 문제를 고쳤습니다.
  - 도시 밖에서 열거나 캐릭터 데이터가 아직 로드되지 않았을 때도, 열 때 캐릭터 정보를 **지연 로드**하도록 변경했습니다.
  - 표시할 캐릭터 정보가 없으면 조용히 실패하는 대신 안내 메시지를 출력합니다.

### 설치 방법
- **기존 넥서스 애드온(nexus_addons ipf 파일)만 삭제**하고 이 애드온을 넣어주세요.
- 혹은 addon manager에서 update 하세요.

### 사용법
- 메뉴를 찾아 nexus addon 아이콘을 클릭합니다.

## 🇺🇸 English

### What's New (v1.0.5)
- **Fixed the Other Character Skill List settings frame not opening**
  - The ⚙ (Call Settings Frame) button in the management list did nothing; it now opens correctly.
  - Character data is now **lazy-loaded** when the frame is opened outside a city or before data has loaded.
  - When there is no character data to show, an info message is displayed instead of silently failing.

### How to Install
- **Remove only your old nexus_addons ipf file**, then add this addon.
- Or update it from the addon manager.

### Usage
- Open the menu and click the Nexus Addons icon.

## 🇯🇵 日本語

### 今回のアップデート (v1.0.5)
- **Other Character Skill List（他キャラスキルリスト）の設定フレームが開かないバグを修正**
  - 管理リストの ⚙（Call Settings Frame）ボタンを押しても開かない問題を修正しました。
  - 都市以外で開いた場合やキャラクターデータ未読込の場合でも、開いた時にキャラ情報を**遅延読込**するようにしました。
  - 表示できるキャラ情報がない場合は、静かに失敗する代わりに案内メッセージを表示します。

### インストール方法
- **古い nexus_addons の ipf ファイルだけを削除**して、このアドオンを入れてください。
- または addon manager から update してください。

### 使い方
- メニューを開き、Nexus Addons のアイコンをクリックします。
